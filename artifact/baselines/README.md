# Baseline Execution Records

This directory holds the execution transcripts, workspace snapshots, and outcome logs for the baseline runs:

* **Condition A (Prompt-Only)**: Standard agent executions where the agent receives the prompt directly and works on the main branch without governance.
* **Condition B (Checklist-Disciplined)**: Runs where the agent writes and updates a `todo.md` list inside its workspace but has no execution boundaries or proof validation requirements.

## File Naming Conventions

* `[task-id]_[run-num]_transcript.txt`: The conversational chat transcript between the user (or orchestrator) and the LLM agent.
* `[task-id]_[run-num]_diff.patch`: The final code diff generated by the agent.
* `[task-id]_[run-num]_outcome.json`: Metadata summarizing the run outcome (e.g., total tokens, completion status, wall-clock time).
* `[task-id]_[run-num]_todo.md` (Condition B only): The checklist document created and modified by the agent.
